# Power Forecast Plotting Guide

This document describes how to generate and improve plots for the substation power forecasts.

## Data Sources

### 1. Forecasts
Forecasts are generated by the `xgb_forecasts` asset and saved as partitioned Parquet files.
- **Path**: `data/power_forecasts/{model_name_and_version}/all_substations.parquet`
- **Schema (`PowerForecast`)**:
    - `valid_time`: Datetime (UTC)
    - `substation_number`: Int32
    - `MW_or_MVA`: Float32 (The predicted value)
    - `nwp_init_time`: Datetime (UTC)
    - `power_fcst_model`: Categorical (e.g., `xgboost_v0.0.1`)
    - `ensemble_member`: UInt8

### 2. Actuals
Actual power data is provided by the `combined_actuals` asset.
- **Source**: Delta Lake table at `data/NGED/delta/live_primary_flows`
- **Columns used**: `timestamp`, `substation_number`, `MW_or_MVA` (coalesced from MW/MVA).

## Current Plotting Implementation

The current implementation resides in `src/nged_substation_forecast/defs/plotting_assets.py`.

### Asset: `forecast_vs_actual_plot`
- **Inputs**: `xgb_forecasts` (DataFrame), `combined_actuals` (DataFrame)
- **Configuration**: `PlotConfig` allows specifying a list of `substation_ids`. If empty, it picks 5 random substations from the forecast.
- **Library**: Altair
- **Output**: 
    - An interactive HTML file saved to `data/NGED/plots/forecast_vs_actual.html`.
    - Metadata logged to Dagster including the plot URL and plotted IDs.

## How to Generate Plots

### Via Dagster UI
1. Launch the Dagster UI: `uv run dagster dev`.
2. Navigate to the `forecast_vs_actual_plot` asset.
3. Click "Materialize".
4. (Optional) Use the "Partition" or "Config" tabs to specify substation IDs.

### Via Python Script
You can materialize the plot asset programmatically:

```python
import dagster as dg
from nged_substation_forecast.defs.xgb_assets import xgb_forecasts
from nged_substation_forecast.defs.metrics_assets import combined_actuals
from nged_substation_forecast.defs.plotting_assets import forecast_vs_actual_plot
from contracts.settings import Settings

settings = Settings()

dg.materialize(
    [xgb_forecasts, combined_actuals, forecast_vs_actual_plot],
    resources={"settings": dg.ResourceDefinition.hardcoded_resource(settings)},
    run_config={
        "ops": {
            "forecast_vs_actual_plot": {
                "config": {"substation_ids": [670104]}
            }
        }
    }
)
```
